# NYC Yellow Taxi Exploratory Data Analysis Mini Project

This repository contains an exploratory data analysis (EDA) project for NYC Yellow Taxi trips using January 2023 trip data. The analysis is implemented in a Jupyter notebook and includes data cleaning, feature engineering, visualizations, and actionable observations.

## Contents

- `nyc_taxi_eda/analysis.ipynb` - Main notebook with data loading, cleaning, analysis, and visualizations.
- `nyc_taxi_eda/yellow_tripdata_2023-01.parquet` - Parquet dataset used by the notebook.
- `nyc_taxi_eda/img/` - Output folder for saved chart images generated by the notebook.

## Key analyses

- Hourly demand patterns and rush-hour peaks
- Tip percentage behavior by distance and payment type
- Payment method impact on tip data quality
- Identification of JFK flat fare trips and outlying fare structures
- Median speed patterns by hour and weekday

## Requirements

- Python 3.10+ (or compatible)
- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
- `missingno`
- `pyarrow` or `fastparquet`

## Setup

1. Create or activate a virtual environment.
2. Install dependencies:

```bash
python -m pip install pandas numpy matplotlib seaborn missingno pyarrow fastparquet notebook ipykernel
```

3. Ensure `yellow_tripdata_2023-01.parquet` is available in `nyc_taxi_eda/`.
4. Launch the notebook:

```bash
jupyter notebook nyc_taxi_eda/analysis.ipynb
```

## Notes

- The notebook includes a dependency check and will attempt to install missing packages in the current kernel.
- Cleaning filters out invalid trips, unrealistic speeds, and bad datetime values.
- Generated plots are saved under `nyc_taxi_eda/img/`.
